# 炉石兄弟个人策略分享

#### 介绍
炉石兄弟吧 4.1 版本兄弟个人策略分享，仅吧友交流使用，侵删。

仅分享策略交流之用，不卖兄弟勿扰。

#### 使用说明
策略完全基于炉石兄弟吧 4.1 版本兄弟，如果未获得激活码爱莫能助。

使用请删除 \Routines\DefaultRoutine 目录，将下载得到的 DefaultRoutine 复制到原 \Routines 目录下

使用卡组请检查卡组内是否包含核心系列卡牌。目前版本兄弟不识别核心卡牌。

请尽量使用英雄初始皮肤（非异画英雄技能）和初始幸运币
